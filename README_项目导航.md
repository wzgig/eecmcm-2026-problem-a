# 项目导航：绿电直连型电氢氨园区优化运行

本项目已完成问题一至问题五的建模、代码求解、结果输出、图表绘制和复盘提升。当前文件夹按“原始资料-公共模块-分问题求解-论文写作资料包”组织。

## 1. 写论文优先阅读顺序

建议论文撰写时优先阅读：

1. `论文撰写资料包/00_论文撰写导航.md`
2. `论文撰写资料包/01_前五问结果总表.md`
3. `论文撰写资料包/02_统一建模口径与符号说明.md`
4. `论文撰写资料包/03_论文图表清单与选图建议.md`
5. `论文撰写资料包/04_模型评价与改进整合稿.md`
6. 各问题文件夹下的 `问题*_完整解答.md`

根目录下的 `A题_建模与分析规划报告.md`、`项目工作要求与进度记录.md`、`全盘复盘与深度优化报告.md`、`前五问统一口径与衔接说明.md`、`模型评价与改进_论文可用稿.md` 均已被整理吸收到 `论文撰写资料包` 中。原文件保留，用于追溯建模过程。

## 2. 目录功能

| 路径 | 功能 | 论文撰写用途 |
|---|---|---|
| `A题.pdf`、附件1-8 | 题面与原始数据 | 数据来源和题目重述 |
| `参考资料/` | 政策、报告、论文资料 | 问题五、参考文献和政策依据 |
| `model_common/` | 公共参数、数据读取、指标计算、绘图风格 | 代码附录和可复现性说明 |
| `问题1_典型风光场景指标分析/` | 刚性满负荷基准 | 发现源荷错配和上网比例超标 |
| `问题2_离散制氨调节优化/` | 离散开停机优化 | 建立离散调节基准 |
| `问题3_连续制氨调节优化/` | 连续负荷率优化 | 量化连续调节相对离散调节的价值 |
| `问题4_离网储能配置优化/` | 离网运行与储能配置 | 评估储能、离网成本和公共电网支撑价值 |
| `问题5_政策与电力系统影响分析/` | 政策和系统影响 | 将前四问数值结果转化为政策建议 |
| `论文撰写资料包/` | 整合后的论文素材 | 最终论文主要参考入口 |

## 3. 保留与不建议直接引用的文件

- `新对话迁移提示词_继续第三至五问.md`：会话迁移记录，不建议放入论文依据。
- `项目工作要求与进度记录.md`：项目过程管理文件，不建议直接放入正文。
- `全盘复盘与深度优化报告.md`：审稿式复盘源文件，已整合为论文可用表达。
- `电工杯论文.docx`、`数学建模格式.docx`：写作格式或草稿文件，数值以脚本输出和 `论文撰写资料包` 为准。

## 4. 复现入口

各问脚本均可从项目根目录运行：

```powershell
python .\问题1_典型风光场景指标分析\solve_q1.py
python .\问题2_离散制氨调节优化\solve_q2.py
python .\问题3_连续制氨调节优化\solve_q3.py
python .\问题4_离网储能配置优化\solve_q4.py
python .\问题5_政策与电力系统影响分析\solve_q5.py
```

所有论文数值应来自各问 `outputs/*.csv` 或 `outputs/*_result_summary.md`，不要手工编写不可复现数据。
