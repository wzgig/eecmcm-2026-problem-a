# 论文撰写导航

本资料包用于最终论文写作，已整合根目录和各问复盘中的重复说明。原始计算文件、CSV、图件和完整解答仍保留在各问题文件夹中，便于复现和查证。

## 1. 五问主线

本文不是五个割裂模型，而是一条逐步增加系统柔性的建模链：

| 问题 | 论文定位 | 核心作用 |
|---|---|---|
| 问题一 | 刚性满负荷基准 | 证明固定制氨负荷下存在购电和上网并存，上网比例超标 |
| 问题二 | 离散开停机调节 | 建立扩容后 72 t/d 系统的离散调节基准 |
| 问题三 | 连续负荷率调节 | 在相同日产量集合下量化连续调节相对离散调节的增量价值 |
| 问题四 | 离网与储能配置 | 删除购售电变量，引入储能，评估完全依赖风光时的成本和可靠性 |
| 问题五 | 系统影响与政策建议 | 用前四问数值证据分析绿电直连园区对电力系统和政策机制的影响 |

## 3. 推荐正文取材顺序

1. 先用 `02_统一建模口径与符号说明.md` 写统一假设、符号和成本口径。
2. 再按问题一至问题五分别压缩 `问题*_完整解答.md` 的“数学模型”和“结果分析”部分。
3. 用 `01_前五问结果总表.md` 核对所有数字，避免不同文件中手工改数造成不一致。
4. 用 `03_论文图表清单与选图建议.md` 选择主图，不必把全部图件放入正文。
5. 最后用 `04_模型评价与改进整合稿.md` 写模型优缺点、推广和进一步研究。

## 5. 写作注意事项

- 吨氨成本统一称为“题面成本口径下的吨氨成本”，不要写成完整工程 LCOA。
- 并网型问题的自发自用电量主口径为 \(E^{self}=E^{re}-E^{sell}\)。
- 离网问题不再使用上网比例，改用弃风弃光率和风光利用率。
- 问题四中 5 MWh 是经济方案，不是消纳最优方案；80 MWh 对应最大弃电场景弃电率不超过 10%，155 MWh 对应零弃电。
- 问题五的政策建议必须引用前四问证据，不要写成泛泛议论文。
