## 当前问题一核心结果

| 指标 | 数值 |
|---|---:|
| 典型日总用电量 | 558.7200 MWh |
| 新能源发电量 | 603.4480 MWh |
| 网购电量 | 172.0438 MWh |
| 上网电量 | 216.7718 MWh |
| 新能源自发自用比例 | 64.08% |
| 总用电量绿电比例 | 69.21% |
| 新能源上网电量比例 | 35.92% |
| 吨氨成本 | 4322.34 元/t |

结论：自发自用比例和绿电比例达标，上网比例不达标。主要原因是制氢制氨负荷全天刚性运行，不能主动匹配风光出力，导致同日既存在购电又存在余电上网。
